# Overview of all Functions

## Create Array

Given an integer size, return array of length size filled with 1s.

#### Example

For size = 4, the output should be
createArray(size) = [1, 1, 1, 1].

#### Input/Output

* **[input] integer size**

A positive integer.

Guaranteed constraints:
1 ≤ size ≤ 1000.

* **[output] array.integer**
